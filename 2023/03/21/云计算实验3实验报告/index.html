利用云平台搭建个人博客

实验目的

熟悉个人博客系统的搭建。

具体包括：
参考方案：注册Github账号，搭建Hexo环境并实现个人博客搭建，撰写实验报告。

硬件环境

联网的计算机一台

软件环境

Linux

实验步骤与内容

创建GitHub账号，并创建与账号对应的github.io repository

本地搭建Hexo环境

  • 使用npm安装hexo-cli
  • 使用hexo init初始化hexo博客，并执行npm install
  • 使用hexo s本地运行hexo博客
    端口一般默认为4000，此时访问http://localhost:4000/得到如下图结果。

将本地Hexo上传到GitHub的repository中

  • 生成SSH Key并添加到GitHub中

  • 修改hexo deploy配置，设置hexo要上传的repository

  • 使用hexo d上传至GitHub的repository

对Hexo进行进一步配置和扩展

  • 数学公式渲染
    数学公式渲染是写博客的一项常用功能。
    从上面的图可以看出，Hexo默认的渲染引擎不支持数学公式渲染。所以需要更换Hexo的renderer为pandoc并配置好mathjax环境。之后博客就可以正常显示mathjax公式了。

  • 更换hexo主题

结论分析与体会

通过这次实验，我搭建了Hexo环境，并实现了个人博客的搭建。成功建立好Hexo后将其上传到个人账户GitHub Pages的repository中。搭建完Hexo博客之后，我还进行了进一步的配置，例如使博客支持数学公式渲染、更换了Hexo的主题等，使博客更加美观，完成更有效的记录。
